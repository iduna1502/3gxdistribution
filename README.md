## まえがき
いづなの3gxは、
アイスステーションZ専用の3gxとなっており、  
__ぽんぽこ様__ __れど様__ __愛芽梨様__ __Clow様__ __西木様__

## 簡単な紹介  
この3gxはより使いやすいものを目指して作成しております。  
moonjumpや座標移動などの基本的なチートはもちろん、レジスタ確認やアドレス監視、計算機や変換器も豊富に取り揃えているので、サーチにも使うことのできる3gxとなっております。
SearchやActionReplayの機能も入ってるので、使いやすいと思います。

## 使い方
#### 導入方法について
この3gxは、CTRPF Ver. 0.7.0 なので、v10.2.1の[boot.firm](https://github.com/Nanquitas/Luma3DS/releases/tag/v10.2.1/)を使用する必要があります。

ダウンロードしたboot.firmは、SDのルートに貼り付け、上書きしてください。
次に、`SD:\luma\plugins\`にzip内の`00040000001BEA00フォルダ`を貼り付けてください。  
3DSを起動するとlumaの設定画面が起動します。  
![lumaConfig](https://user-images.githubusercontent.com/80764644/126063433-630332e3-e469-499a-b054-43cd68dcb4bc.jpg)  
`Enable game patching `にチェックを入れてください。  
startボタンを押し、3dsを起動してください。  
ホーム画面が起動したら、DD + L + select でRosalinaMenuを起動してください  
plugin loader というのがあるので、Enabled にしてください。  
あとは、アイスステーションZを起動し、起動時に上画面が一瞬水色になったら成功です。  
初回起動時にPIN入力を要請されます。  
初回起動時のPINは、`1234`です。  
  
#### PINについて  
![pin](https://user-images.githubusercontent.com/80764644/126063740-d4c956e3-8cab-4b1f-9311-aa82aacaa912.png)  
PINのついているコードは、PINを解除しない限り使用することができないようになっています。  
PINは一般公開されていないので、いづなのDMに来てください。

## おまけ
上で紹介した機能のほかに、CTRPFには便利な機能がほかにもいっぱいあるので、それを紹介していきます。  
#### サウンド追加
CTRPFでカーソルを動かしたときなどに、音を鳴らす機能があります。  
音を鳴らすための[CTRPF_DefaultSounds.zip](https://github.com/iduna1502/3gxdistribution/files/6837113/CTRPF_DefaultSounds.zip)をダウンロードします。  
ダウンロードしたら解凍し、3gxと同じ階層にコピーしてください。  
#### Guide追加
GWの条件を記したtxtを作ったので、ぜひ入れてみてください。  
導入の仕方は[Guide.zip](https://github.com/iduna1502/3gxdistribution/files/6837116/Guide.zip)をダウンロードし、3gxと同じ階層にコピーしてください。  
CTRPFのメニューから、Guideという場所を押すと、GW条件コードというのが表示されるので、それを選択してください。
